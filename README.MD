![Login](preview%5CAnimation.gif)


## Run the app in localhost

### React App:

```bash
cd client
npm install
npm start
```

### Node App:

```bash
cd server
npm install
npm run dev-server
```

### Variables in .env file:
* `DATABASE` - database connection string
* `SECRET_TOKEN` - secret token for JWT
